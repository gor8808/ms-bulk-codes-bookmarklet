# Установка и использование bookmarklet для МойСклад

## 1) Установка (один раз)

1. Откройте файл `bookmarklet.url.txt`.
2. Скопируйте всю строку, которая начинается с `javascript:`.
3. В браузере Chrome/Edge:
   - Покажите панель закладок (`Ctrl+Shift+B` / `Cmd+Shift+B`).
   - Создайте новую закладку (ПКМ по панели закладок -> "Добавить страницу...").
   - Название: `MS Bulk Codes`.
   - URL: вставьте строку из `bookmarklet.url.txt`.
4. Сохраните закладку.

![Пример создания закладки](/Users/gorgrigoryan/Documents/New project/screenshots/install-bookmark-example.png)

## 2) Использование

1. В МойСклад откройте документ "Ввод в оборот кодов маркировки" в режиме редактирования.
2. Нажмите закладку `MS Bulk Codes`.
3. В окне "Массовое добавление кодов маркировки":
   - Вставьте коды (по одному в строке).
   - Нажмите "Добавить".
4. Дождитесь завершения. Во время работы видно прогресс и кнопку "Стоп".
5. В конце откроется итог:
   - `Добавлено` — успешно добавленные коды.
   - `Ошибки` — коды, которые МойСклад не принял.
   - `Пропущено` — пустые строки и коды, не обработанные после нажатия "Стоп".
6. Если есть ошибки, нажмите "Скопировать ошибки" и запустите повторно только для этих кодов.

## 3) Что делать при проблемах

1. Убедитесь, что вы на странице документа `.../app/#enrollorder/edit?...`.
2. Если МойСклад обновился и bookmarklet перестал работать, обратитесь к владельцу скрипта (Owner X) и передайте текст ошибки/скриншот.
3. Не сохраняйте документ, если тестовые строки добавились некорректно; закройте документ без сохранения.
