# Расширение для массовых кодов МойСклад

Это расширение использует путь, который недоступен bookmarklet:

- `chrome.debugger`
- `Input.insertText`
- `Input.dispatchKeyEvent`

Текущая версия сама ищет рабочее поле на странице МойСклад:

1. Откройте страницу `Ввод в оборот` в МойСклад.
2. Откройте popup расширения.
3. Вставьте коды по одному в строке.
4. Нажмите `Добавить`.

## Что умеет

- массовая отправка кодов
- прогресс по каждому коду
- остановка процесса
- итоговая сводка
- копирование ошибочных кодов

## Как загрузить локально

### Вариант 1: скачать готовый ZIP из GitHub Release

1. Откройте страницу релиза:
   [v0.1.0](https://github.com/gor8808/ms-bulk-codes-bookmarklet/releases/tag/v0.1.0)
2. Скачайте файл `masovye-kody-moysklad-v0.1.0.zip` из блока `Assets`.
3. Распакуйте ZIP в любую папку на компьютере.
4. Откройте `chrome://extensions`.
5. Включите `Developer mode`.
6. Нажмите `Load unpacked`.
7. Выберите папку `extension` из распакованного архива.

### Вариант 2: из репозитория

1. Откройте `chrome://extensions`.
2. Включите `Developer mode`.
3. Нажмите `Load unpacked`.
4. Выберите папку `extension`.

## Примечания

- Первый код может очищать поле через `Ctrl+A` + `Backspace`.
- Между кодами выдерживается пауза.
- Расширение работает только на вкладке документа `enrollorder/edit`.
